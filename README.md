## Packaging and publishing
1. Install vsce.
```
npm install -g vsce
```

2. Package files.
```
vsce package -o extensions/
```

3. Publish changes
```
vsce publish.
```

See [this guide](https://code.visualstudio.com/api/working-with-extensions/publishing-extension) for more.
